
Development
====================

Source Code Repository
----------------------------

The source of |managesieve| is maintained at
`GitLab <https://gitlab.com>`_. Patches and pull-requests are hearty
welcome.

* You may browse the current repository at the
  `Repository Browser
  <https://gitlab.com/htgoebel/managesieve>`_

* Or you may check out the current version by running::

    git clone https://gitlab.com/htgoebel/managesieve.git

Issue Tracker
----------------

Please submit bugs and enhancements to the Issue Tracker at
<https://gitlab.com/htgoebel/managesieve/issues>.


Historical Note
-------------------

|managesieve| was hosted at origo.ethz.ch, but this site closed
at May 2012. Then |managesieve| was hosted gitorious.org,
which was closed and merged into gitlab.com in May 2015.
